# GPT Assistants Playground

Open-source Gradio application that mirrors the OpenAI Assistants UI while adding hooks for local actions, logs, and assistant management features.

## Summary

- Ships with installers plus scripts that spin up a local experience for creating assistants, attaching tools, and observing logs.
- Extends the stock Assistants interface with agent managers, assistant databases, and the ability to execute actions on the local machine (with permissions).
- Used in *AI Agents in Action* to explore running assistants under [[Agentic Behavior Trees]] and supervising local code execution.

## Connections

- Serves as a testbed before publishing GPTs described in [[GPT Assistants]].
- Enables instrumentation aligned with [[AgentOps]] and [[Reasoning Strategies]].
