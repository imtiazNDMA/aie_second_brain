# ZeRO

**Entity Type:** Framework, Technology  
**Category:** Machine Learning Training Optimization  
**Known For:** Memory optimization for training large models

## Summary

ZeRO (Zero Redundancy Optimizer) is a framework developed by Microsoft for optimizing memory usage during the training of large deep learning models. It eliminates redundancy in optimizer states, gradients, and parameters across data parallel processes, enabling training of much larger models on existing hardware. It was mentioned in the context of advanced frameworks for LLM training infrastructure.
