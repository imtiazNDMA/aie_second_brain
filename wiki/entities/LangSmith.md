# LangSmith

## Summary
Platform by LangChain for debugging, testing, deploying, and monitoring AI workflows. Provides tracing, evaluation tools, and integration with LangGraph Platform.

## Key Features
- **Debugging** — trace LLM calls, tool usage, and agent steps
- **Testing** — evaluate prompt quality and model responses
- **Monitoring** — track production metrics and catch errors
- **Integration** — works with LangGraph Platform for agent deployment
