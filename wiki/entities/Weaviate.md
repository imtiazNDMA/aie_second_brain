# Weaviate

## Summary

Open-source, cloud-native vector database that stores dense embeddings alongside structured metadata. Supports hybrid keyword + vector search, module plug-ins (OpenAI, Cohere, Hugging Face), and multi-tenant deployments, making it a common backbone for modular or enterprise-scale [[Retrieval-Augmented Generation]] stacks.

## Connections

- Pairs with [[LangChain]] and [[LlamaIndex]] retrievers for semantic search pipelines.
- Alternative to managed stores like [[Pinecone]] or lightweight options such as [[Chroma]].
- Recommended in [[2026-04-12-14-types-of-rag]] for modular and graph-style RAG implementations.
