# LangGraph

LangChain’s graph-based framework for composing multi-agent workflows with explicit state machines, streaming, and persistence.

## Highlights

- Featured in *Building Agentic AI Systems* for orchestrating Coordinator/Worker/Delegator roles.
- Supports guardrails, tool routing, and memory persistence via structured graphs.
