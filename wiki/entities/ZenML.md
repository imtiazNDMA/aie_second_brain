# ZenML

Open-source MLOps orchestrator used in *LLM Engineer’s Handbook* to manage pipelines, stacks, metadata, and reproducible LLM experiments.

## Highlights

- Supports stack components (orchestrators, artifact stores, model registries) and integrates with Comet, Hugging Face, AWS.
- Enables [[Continuous Training]] and deployment workflows for the [[LLM Twin]] project.
