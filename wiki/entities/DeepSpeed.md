# DeepSpeed

**Entity Type:** Framework, Technology  
**Category:** Machine Learning Training Optimization  
**Known For:** Efficient training of large-scale models

## Summary

DeepSpeed is a deep learning optimization library developed by Microsoft that makes distributed training easy, efficient, and effective. It provides technologies like ZeRO (Zero Redundancy Optimizer) for memory optimization, pipeline parallelism, and other techniques to train models that are orders of magnitude larger than what is feasible with standard approaches. It was mentioned in the context of advanced frameworks for LLM training infrastructure.
