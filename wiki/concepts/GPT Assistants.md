# GPT Assistants

## Definition

Customizable assistants built with OpenAI’s GPT platform that combine instructions, conversation history, files, and actions to solve user tasks.

## Capabilities

- Builder workflows support defining personas, delimiters, and guardrails (“Culinary Companion”, “Calculus Made Easy GPT”).
- Attachments: Code Interpreter for data science, file uploads for bespoke knowledge, and GPT Store publishing for distribution.
- [[Custom Actions]]: FastAPI endpoints described via OpenAPI specs, reachable through ngrok or hosted HTTPS services.

## Related Concepts

- [[Custom Actions]] — Mechanism for extending assistants with real-world APIs.
- [[Prompt Flow]] — Used to systematize and evaluate prompt instructions before publishing.
