# Mixture Density Network

## Definition

Neural network that outputs parameters of a mixture distribution (e.g., Gaussian mixture) instead of point estimates, enabling modeling of multi-modal targets.

## Usage

- Supports amortized [[Variational Inference]] and probabilistic regression tasks (chapter 11).
