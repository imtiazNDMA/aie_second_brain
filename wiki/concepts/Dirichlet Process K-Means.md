# Dirichlet Process K-Means

## Definition

Nonparametric extension of K-means that allows a potentially infinite number of clusters via a Dirichlet process prior, letting data dictate component counts.

## Use Cases

- Adaptive clustering when the number of topics/entities is unknown.
- Serves as a bridge between classical clustering and Bayesian mixture models.
