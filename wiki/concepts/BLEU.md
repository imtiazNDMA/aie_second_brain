# BLEU

## Definition

Bilingual Evaluation Understudy (BLEU) is an n-gram overlap metric that compares generated text against one or more references with a brevity penalty.

## Why It Matters

- Provides a fast, standardized automatic score for translation and generation quality.
- Useful for trend comparison across model or prompt variants when paired with human review.
