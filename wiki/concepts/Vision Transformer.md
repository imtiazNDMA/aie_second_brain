# Vision Transformer

## Definition

Computer-vision architecture that applies transformer encoders to image patches treated as token sequences.

## Why It Matters

- Reuses attention-based design patterns from NLP for image classification.
- Scales well with data and model size while enabling flexible global context modeling.
