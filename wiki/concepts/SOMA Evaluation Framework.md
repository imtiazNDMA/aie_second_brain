# SOMA Evaluation Framework

## Definition

Prompt-evaluation rubric from *[[Prompt Engineering]] for LLMs* that scores outputs across Subject, Objective, Medium, and Audience dimensions.

## Usage

- Provides structured offline tests for new prompt profiles before deployment.
- Can be automated via LLM graders or human reviewers and tracked over time for regression detection.
