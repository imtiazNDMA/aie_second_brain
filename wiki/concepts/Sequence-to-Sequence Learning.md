# Sequence-to-Sequence Learning

## Definition

Modeling setup that maps an input sequence to an output sequence, commonly using encoder-decoder structures with attention.

## Why It Matters

- Supports translation, summarization, transcription, and structured generation tasks.
- Requires careful handling of masking, variable lengths, and decoding policies.
