# AI Interface

## Definition

The conversational layer that replaces GUIs and APIs with natural-language exchanges between users and agents. Chapter 1.4 (“Peeling back the AI interface”) highlights how prompts, RAG inserts, and persona cues form a new UX surface.

## Key Points

- Interfaces can be direct (ChatGPT), proxy (agents acting on behalf of tools), assistant (GPTs with instructions and files), or autonomous (crews executing plans end-to-end).
- Good interfaces expose constraints (available tools, safety rules) so users can predict agent behavior.
- Supporting artifacts include prompt templates, persona briefs, and evaluation rubrics to assure reliability.
