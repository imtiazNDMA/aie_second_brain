# Model Evaluation

Assessing model performance on held-out data to estimate generalization.

## Definition

Metrics and methods for evaluating ML models: accuracy, precision, recall, F1, AUC, [[Perplexity]], [[BLEU]], [[ROUGE]], etc. Goes beyond metrics to include human evaluation and ablations.
