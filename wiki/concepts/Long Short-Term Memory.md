# Long Short-Term Memory

## Definition

Recurrent architecture with input/forget/output gates and a cell state designed to preserve long-term dependencies.

## Why It Matters

- Reduces vanishing-gradient issues in longer sequences.
- Widely used baseline for sequence tasks before transformer dominance.
