# Agent Trust and Safety

## Definition

Practices that make autonomous agents trustworthy: transparency, explainability, uncertainty reporting, permissioning, and ethical governance.

## Techniques

- Instrumentation of tool calls, approvals, and memory writes for auditability.
- Explainability layers (rationales, self-critiques) that expose reasoning to humans.
- Safety policies enforcing least privilege, runtime approvals, and red-team evaluations.
