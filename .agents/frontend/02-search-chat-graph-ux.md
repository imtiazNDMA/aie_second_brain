# Search, Chat, and Graph UX

## Search UX

- Show source type, relevance, and recency indicators.
- Highlight matched terms and concepts.
- Provide filters: type, tags, source date, confidence.
- Offer query suggestions from top entities/concepts.

## Chat UX

- Split view: answer panel and evidence panel.
- Stream answer text while citations appear independently.
- Clicking a citation opens source page and scrolls to evidence.
- Show uncertainty states when evidence is weak.

## Graph UX

- Start with neighborhood around selected node.
- Use edge labels for relation hints when available.
- Keep controls simple: depth, node type toggles, reset.
- Include list fallback for users who do not prefer graph canvas.

## Interaction Patterns

- Preserve user context when switching between search, chat, and page view.
- Keep selected query and filters sticky across navigation.
- Support deep-link URLs for search query, page slug, and graph node.

## Visual Direction

- Avoid generic monochrome layouts.
- Use a clear type system and spacing rhythm.
- Emphasize readability for long-form markdown and citations.
